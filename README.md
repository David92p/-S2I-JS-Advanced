## API Reference

#### Get City

```http
    https://api.teleport.org/api/cities/?search={city_name}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `city_name`      | `string` | **Required**. city name to fetch |


## Installation

Install -S2I-JS-Advanced
 with npm

```bash
  git clone 'https://github.com/David92p/-S2I-JS-Advanced.git'
  npm install 
  npm run build
```
