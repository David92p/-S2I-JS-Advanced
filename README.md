## API Reference

#### Get City

```http
    https://api.teleport.org/api/cities/?search={city_name}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `city_name`      | `string` | **Required**. city name to fetch |
